Sou Ashley, desenvolvedor fullstack baseado em Moçambique. Combino engenharia sólida com visão de produto para criar soluções que resolvem problemas reais — desde sistemas POS para negócios locais até plataformas e-commerce completas.

Estudo Programação Web no ITC (Instituto de Transportes e Comunicações) enquanto construo projectos independentes com stack moderno: Node.js, React, React Native, Electron, MySQL e mais.

Tenho interesse especial em fintech, plataformas para o mercado africano e integração de pagamentos locais como M-Pesa e e-Mola.
